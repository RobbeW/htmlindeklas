# Contentstructuur HTML in de Klas

Deze map bevat de lesinhoud voor de platformpagina.

- `catalog.json`: overzicht van hoofdstukken en programmeer-/webconcepten.
- `exercises.js`: oefeningen, startcode, controles en feedbackmetadata voor de browserapp.

De app laadt `exercises.js` rechtstreeks als script. Dat houdt de tool bruikbaar als statische site en vermijdt een zware buildstap.
