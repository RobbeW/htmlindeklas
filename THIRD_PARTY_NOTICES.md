# Third-party notices

HTML in de Klas vendors a small set of browser libraries locally so the classroom app can run without relying on CDN availability during a lesson.

## CodeMirror 5.65.14

Used for the HTML editor.

Source: https://codemirror.net/  
Package CDN source used for vendoring: https://cdn.jsdelivr.net/npm/codemirror@5.65.14/

## Marked 12.0.2

Used to render Markdown exercise descriptions.

Source: https://marked.js.org/  
Package CDN source used for vendoring: https://cdn.jsdelivr.net/npm/marked@12.0.2/

## jsPDF 2.5.1

Used for individual and batch PDF export.

Source: https://github.com/parallax/jsPDF  
Package CDN source used for vendoring: https://cdn.jsdelivr.net/npm/jspdf@2.5.1/

## HTMLHint 1.8.0

Used for HTML linting and structure feedback.

Source: https://htmlhint.com/  
Package CDN source used for vendoring: https://cdn.jsdelivr.net/npm/htmlhint@1.8.0/
